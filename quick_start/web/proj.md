# 创建工程

首先我们使用 IDE 创建一个符合 maven 标准的 web 工程，创建后目录结构如下：

> 注：如果不清楚如何通过 IDE 创建标准的 maven web 工程请先自行了解。

```
users
|----src
|    |----main
|    |    |----java
|    |    |    |----hello
|    |    |----resources
|    |    |    |----conf
|    |              |----beans.xml
|    |              |----config.xml
|    |    |----webapp
|    |         |----WEB-INF
|    |              |----web.xml
|    |----test
|         |----java
|         |----resources
|----pom.xml
```
